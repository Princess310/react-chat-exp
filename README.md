### Setup
1. to setup project, there are two ways:
```
npm run setup
or
npm install
```
2. to run project:
```
Dev: npm run start
Pro: npm run build
Test: npm run test
```
3. Note:
```
This is a complex React Web App, which is started by [react-boilerplate](https://github.com/react-boilerplate/react-boilerplate.git).
```
